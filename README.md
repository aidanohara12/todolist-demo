# todolist-demo

A small React + Vite + Tailwind todo list app.

## Prerequisites

- **Node.js** 18 or newer (check with `node -v`)
- **npm** (ships with Node)

If you don't have Node, install it from [nodejs.org](https://nodejs.org/) or via a version manager like `nvm`.

## Setup

```bash
git clone <repo-url>
cd todolist-demo
npm install
```

`npm install` pulls in the dependencies declared in `package.json`:

- Runtime: `react`, `react-dom`
- Dev: `vite`, `@vitejs/plugin-react`, `tailwindcss`, `@tailwindcss/vite`

## Run

```bash
npm run dev
```

Then open the URL Vite prints (usually http://localhost:5173).

## Other commands

- `npm run build` — build for production into `dist/`
- `npm run preview` — serve the production build locally
